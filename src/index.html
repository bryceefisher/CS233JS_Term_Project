Bryce's Brewery Finder Instructions

Instructions for the Brewery Finder App:

  1. Enter the City Name:

    Type the name of the city in the input field provided.

    The city name must match a value from the suggested list.

  2. Enter the State Code:

    Type the two-letter state code (e.g., CA for California) in the input field provided.

    The state code must match a valid two-letter code from the suggested list.

  3. Enter the Number of Breweries:

    Type the desired number of breweries to be returned (maximum 30) in the input field provided.

  4. Click the "Submit" Button:

    After entering the city name, state code, and number of breweries, click the "Submit" button.

    The app will retrieve and display the requested breweries on the embedded map.

  5. Explore the Map and Brewery List:

    On the embedded map, you will see icons representing the breweries.

    Click on a beer icon on the map or a brewery in the list to open an info window.

    The info window will display the brewery's name, address, phone number, and website URL.

Cheers!
